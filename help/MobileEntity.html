MobileEntity


superclasses: Entity


Mobile entity is an entity that can be updated. In the update method the position of 

the entity is updated according to a force 'requested' from the controller by calling

to it the . getForce method. Create the controller and assign it to the controller variable

at the init method.


*new(world, position, radius, mass, velocity)

world is an instance of EntityManager or EntityManager2. There is an added variable for velocity. 


update

update the entity. Currently using Euler integration. If you need a better integrator make a subclass and

code the update method.
