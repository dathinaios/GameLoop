Entity



Entity is the base class for all entity classes. 


See also: MobileEntity, Vehicle



*new(world, position, radius, mass)

world is an instance of EntityManager or EntityManager2


add

add the entity to the world. This method is called when the entity is activated so

you should write your own if you use EntityManager2. For example:


add{ world.addMob(this) //add a mobile entity

}


remove

remove the entity from the world. As with add you should write your own if you 

use EntityManager2. For example:


remove { world.removeMob(this); //remove the mobile entity

  reps.do{arg i; i.remove};

}


attach

ratach a representation to the entity. Multiple representations can be attached and are added to the list in 

the reps variable. They should be removedonwhen .remove is called as above


collision

the method called when a collision is detected


activate

This method is the last to be called. Activating the entity calls the add methods to the entity

as well as activating the representations.
